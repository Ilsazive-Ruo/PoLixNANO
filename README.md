# POLixNANO

## Requrements:

* python	3.9
* joblib
* RDkit
* numpy
* pandas
* scikit-learn
* xgboost

## Usage
ML_RNA.py: script to fit models for prediction of physicochemical properties of nanoparticles.
transfer_fit.py: script to fit equation for prediction of transfection efficiency of nanoparticles.
